Before responsive design (popularized in 2011-2012), most websites were built only for desktop. When smartphones became popular around 2007-2010, these sites were nearly unusable on mobile. Users had to:

  • Pinch and zoom constantly
  • Scroll horizontally AND vertically

  • Read microscopic text

üí° What Changed?

Responsive Design was introduced by Ethan Marcotte in 2010 and became mainstream around 2012-2014. Key innovations included:

  • Media Queries: CSS that adapts to screen size
  • Flexible Grids: Layouts that scale proportionally
  • Fluid Images: Pictures that resize automatically
  • Mobile-First Design: Building for phones first, then desktop
  • Touch-Friendly Buttons: Big enough to tap with fingers

Today, over 60% of web traffic is mobile. A non-responsive site in 2026 would be considered completely broken!
